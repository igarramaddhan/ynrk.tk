# [ynrk.tk](https://ynrk.tk/)

Personal blog created with [gatsby](https://www.gatsbyjs.org/)

#### Run Locally

1. **Clone**
    ```sh
    git clone https://github.com/igarramaddhan/ynrk.tk.git
    ```

1.  **Install dependency**

    Navigate into the cloned directory
    ```sh
    cd ynrk.tk/
    yarn
    ```

1.  **Start the server**
    ```sh
    yarn dev
    ```
